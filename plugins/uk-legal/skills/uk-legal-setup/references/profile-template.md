# Practice-profile template

The `uk-legal-setup` skill writes this file to `~/.claude/uk-legal-profile.md`. The `legal-research` and `legal-debate` skills read it (if present) to default jurisdiction, silk-system framing, and citation style. Fields the user left for later carry `[PLACEHOLDER]`.

```markdown
---
default_jurisdiction: England & Wales
silk_system: true
citation_style: OSCOLA
---

# uk-legal practice profile

- **Default jurisdiction:** England & Wales
  (the jurisdiction downstream skills assume unless a question states otherwise)
- **Silk system (KC/SC):** yes
  (drives legal-debate advocate personas — King's Counsel / Senior Counsel)
- **Primary practice areas:** [PLACEHOLDER]
- **Citation style:** OSCOLA
- **Court-tier focus:** [PLACEHOLDER]

_Generated by the uk-legal-setup skill. Last updated: <YYYY-MM-DD>._
```

## How consuming skills use it
- **Jurisdiction / silk system:** if `silk_system` is false or the jurisdiction is non-silk, `legal-debate` flags that the KC/SC framing falls outside tailoring before spawning advocates.
- **Citation style:** `legal-research` and `legal-debate` format authorities in the recorded style (default OSCOLA).
- If the file is absent, the skills behave exactly as before (England & Wales / silk defaults, ask when it matters).
